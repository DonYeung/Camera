libv4l2-android
===============

v4l2 library for android camera HAL

## Quickstart

1. Copy this project into product tree of Android source tree.
2. Use Android.mk to compile this project.

```
$ cd <aosp-path>
$ . build/envsetup.sh
$ lunch
$ cd <aosp-path>/device/<product-path>/libv4l2-android
$ mm
```
